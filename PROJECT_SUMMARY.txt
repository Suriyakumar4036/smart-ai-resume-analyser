1. CONTEXT: WHY THE ORIGINAL PROJECT WAS REPLACED
--------------------------------------------------------
The original Python/Streamlit repository ("AI-Resume-Analyzer") was technically deprecated and fundamentally broken on modern systems. As an AI Agent, you should NOT attempt to fix the old code. We have deleted the reliance on the old stack for the following reasons:

- SPACY INCOMPATIBILITY: The project required Spacy v2.3.5, which is no longer compatible with modern Python environments (3.11+) without massive hacks.
- BROKEN LIBRARIES: It relied on 'pyresparser', which is an unmaintained library that hardcodes outdated Spacy models and throws 'TypeError: add() takes exactly 2 positional arguments' due to API changes in Spacy 3.
- ENVIRONMENT FRAGILITY: The old project required complex local setups, C++ build tools for Spacy, and specific database (MySQL) configurations that made it impossible for non-technical users to run.

NEW ARCHITECTURE (NEXUS AI):
We have built "Nexus AI" as a 100% modern, robust replacement.
- TECH STACK: Next.js 14 (App Router), React 19, Tailwind CSS 4.
- ANALYSIS ENGINE: A custom, high-accuracy ATS scoring engine that evaluates resumes based on:
    * Quantifiable Impact (Number of metrics/percentages)
    * Professional Action Verbs
    * HR Red Flags (Personal pronoun detection)
    * Keyword Density (12-15 targeted hard skills)
    * Structural Integrity (Education, Experience, Summary sections)
- EXTRACTION: High-stability PDF parsing using 'pdf-parse-fork'.
- PORTABILITY: Runs on any machine with Node.js; no complex database or Python setup required.
